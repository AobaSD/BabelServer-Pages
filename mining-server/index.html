Babel Server 採掘鯖

このページでは、Babel Server 採掘鯖の特徴を紹介しています！

特徴

Babel Server 採掘鯖には主に、以下の特徴があります！

  • 多彩な採掘場 - 洞窟、ネザー、バニラ、エンド、海、海底神殿、終焉、黄金、天界といった、様々な採掘場が存在します。
  • 取引システム - お金の取引が簡単に出来るようになっているため、他のプレイヤーとアイテムを取引することもできます！
  • ランキング - 稼いだ金額やブロック破壊数などでランキングが表示されます。上位を目指してみませんか？
  • クラフト - サーバー内でアイテムを消費して新しいアイテムを作ることもできます！
  • 参加方法 - 採掘サーバーへの参加方法を確認しましょう

多彩な採掘場

洞窟、ネザー、バニラ、エンド、海、海底神殿、終焉、黄金、天界といった、様々な採掘場が存在します。
各エリアで出現するアイテムは、以下をクリックして確認してください。

  1. 洞窟 - Lv0 以上
  2. ネザー - Lv10 以上
  3. エンド - Lv25 以上
  4. 海 - Lv50 以上
  5. 海底神殿 - Lv100 以上
  6. 終焉 - Lv300 以上
  7. 洞窟 - 採掘数 6,000,000 以上
  8. 天界 - 採掘数 10,000,000 以上

取引システム

お金の取引が簡単に出来るようになっているため、他のプレイヤーとアイテムを取引することもできます！
Babel Serverでは毎日、多くのアイテム・Babel Moneyが取引されており、上位層になるための重要な要素でもあります。
そのため、コマンドまたはアイテムを使って、簡単に送金をすることができます！

ランキング

Babel Server では、所持金・採掘数のランキングが掲示されています。
ランキングの上位になり、プレイヤーたちに権威を誇示しましょう😃

  • 現在のランキングは、ここをクリックして確認できます。

クラフト

サーバー内で掘ったアイテムの使い道として、売却するのもありですが、クラフトでアイテムを作ることも可能です！
たとえば...

  • 新しいピッケル
  • 価値の高い素材
  • エフェクトの付く防具

など、様々なアイテムを作成することができます！
